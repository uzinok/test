BOLT-SLIDER набор плагинов для создания слайдеров на сайте.

  • Описание проекта

  • Простой слайдер. Слайдер с минимальным количеством опций.

  • Простой слайдер. Зацикленный.

Описание.

  • Стили только необходимые для работы слайдера. В библиотеке нет стилей для кнопок навигации и пагинации. Элементы управления могут располагаться в любом месте html кода. Для создания кнопок пагинации передается блок в котором они будут генерироваться.
  • Если не переданы или с ошибкой переданы опции в консоль выводинтся ошибка:
    ⚠️ Not slider options
⚠️ didn't get html element: slider
⚠️ didn't get html element: sliderPrew
⚠️ didn't get html element: sliderNext
⚠️ didn't get html element: paginationWrap
⚠️ didn't get html element: slider list
⚠️ didn't get html element: slider wrap list
⚠️ didn't get html element: slider items
⚠️ didn't get html element: slider contents
  • Управление слайдером кнопками навигации и пагинации (если для пагинации передан тег button), на сенсорных устройствах касанием.
  • При наведении курсора на слайдер и при попадании элементов слайдера в фокус автоматическое пролистывание слайдера останавливается до ухода курсора со слайдера. При клике на слайдер так же останавливается автоматическое пролистывание слайдера.
  • Из-за доступности изначально есть адаптивная высота.

Верстка.

  • .bolt-slider - сам слайдер

  • кнопки вперед и назад могут быть в любом месте разметки

  • .bolt-slider__list-wrap - данный класс необходим для правильной работы слайдера

  • .bolt-slider__list - Список слайдов, данный класс необходим для правильной работы слайдера

  • .bolt-slider__item - элемент списка слайдов, следит за появлением самих слайдов для скрин-ридера, данный класс необходим для правильной работы слайдера

  • .bolt-slider__content - сам слайд, когда родитель не активный (не содержит класс .bolt-slider__item--active), содержимое скрыто (display: none), данный класс необходим для правильной работы слайдера

  • .bolt-slider__pagination блок для пагинации, класс может быть любой, сам блок может находиться в любом месте разметки

Доступность.

  1. для слайдера необходимо добавить атрибут 'aria-label'.
    Этот атрибут не должен содержать слово слайдер или карусель. так как скрин-ридер озвучит 'aria-roledescription', значение которого можно изменить опционально. По умолчанию 'carousel'.

  2. слайдам необходимо добавлять атрибут 'aria-label'.
    Так же в 'aria-label' не нужно прописывать слово слайд так как скрин-ридер озвучит 'aria-roledescription', значение которого можно изменить опционально. По умолчанию 'slide'.
